# eshop

---

Requirements:
- PHP >= 8.4
- NPM

---

How to run:

1. Clone the repository to your PC: `git clone git@github.com:ac1dmarv3l/eshop.git`
2. Go to the directory: `cd eshop`
3. Adjust `.env` file to your configuration. Use `.env.example` as a reference.
4. Install dependencies: `npm i` and `composer install`
5. Start PHP dev server: `php -S localhost:8010 -t .`
6. Start Vite dev server: `npm run dev`
