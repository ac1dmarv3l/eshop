# eshop

---

Technologies used:
- Symfony 8
- TailwindCSS
- Alpine.js
- Vite

---

How to run for development:

1. Clone the repository to your PC: `git clone git@github.com:ac1dmarv3l/eshop.git`
2. Go to the directory: `cd eshop`
3. Create and adjust `.env.local` file to your configuration. Use `.env.example` as a reference.
4. Install dependencies: `npm install` and `composer install`
5. Start PHP dev server: `symfony server:start` or `php -S localhost:8000 -t public`
6. Start Vite dev server: `npm run dev`
